Detailed Description

Block controller for a DMA FIFO block.

The DMA FIFO block has the following features:

  • One input- and output-port (type agnostic)
  • Configurable base address and FIFO depth
  • The base storage for the FIFO can be device specific. Usually it will be an off-chip SDRAM bank.
